Understanding Django Template Engine

Django uses its own template engine to render dynamic content. Let's explore some key features:

Blocs

Blocs allow you to define reusable sections in templates. They are defined using the {% block %} tag and can be overridden in child templates using the {% block %} tag with the same name.

Boucles For

Django templates support {% for %} loops which iterate over lists or querysets. For example: 

            
{% for item in items %}
    {{ item }}
{% endfor %}

Structures de Contrôle If

With {% if %} statements, you can add conditional logic to your templates. For example: 

            
{% if condition %}
    Content to show when condition is true.
{% else %}
    Content to show when condition is false.
{% endif %}

Affichage des Variables Passées par Contexte

Variables passed by context are displayed using double curly braces {{ variable }}. For example: 

            
{{ user.username }}
